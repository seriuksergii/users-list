Installation

To run this project locally:

1.	Clone the Repository:

         https://github.com/seriuksergii/react-todo.git

2.	Navigate to the project directory:

         cd react-todo

3.	Install dependencies:

          npm install

4.	Start the application:
 
          npm run build
